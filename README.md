# Hori-Api

## ```INSTALL And UNINSTALL```

> npm install hori-api
>  
> npm uninstall hori-api


## ```Otakudesu```
``` 
const hori = require('hori-api');
const judul = 'kaifuku'

hori.otakudesu(judul)
    .then(result => {
     console.log(result)
});
```
